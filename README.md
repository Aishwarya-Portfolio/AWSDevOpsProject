# AWSDevOpsProject
Deployement of Nginx webpage in EC2 instance using AWS DevOps.

**Tools Used -->**
	Source Code Repository - Code Commit
	Build - Code Build
	Deploy - Code Deploy
	CI/CD - Code Pipeline
	Access management - IAM
	Artifact Storage - S3
	Deployed to - EC2 instance (Ubuntu)
